# KiCad state workflow

## Overview

This document describes the versioned KiCad state workflow used by the hardware domain to keep the KiCad user environment reproducible inside the devcontainer.

The versioned KiCad state is stored under `tools/kicad_state/state/`.

The underlying scripts are documented in [../../../../tools/kicad_state/kicad_state.md](../../../../tools/kicad_state/kicad_state.md).
